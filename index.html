Design

Modern, Intuitive, Configurable and Free to use

iFL03 is a continuation of the iRon project with a refreshed visual style, new features and overlays, and a friendlier setup experience via a built‑in Chromium (CEF) UI. It runs well on modest PCs and keeps dependencies to a minimum.

Install & run

  1. Download the latest installer exe from Releases and install anywhere writable.
  2. Launch iFL03. Start iRacing; overlays show when you’re in the car.
  3. Use Preview Mode to place overlays. Tweak options in the UI or config.json.
