iFL03 OVERLAYS

iRACING OVERLAYS FOR EVERYONE

Download Install Guide
Overlays running in iRacing Live capture from an active iRacing session using the iFL03 overlays. Performance Mode Caps all overlays to 30fps and also allows for per overlay fram cap. About 4% extra CPU Usage Measured on test rig with i7-10700k, iFL03 typically adds roughly 4% CPU with all overlays on, in an iRacing Session. Online & AI racing Footage recorded in 30fps but shows the fps counter. Scenario: 20+ cars on track in online practice session. Multi Class Support All relevant overlays support multi class racing with class colors and class position.

*Usernames are blurred for privacy reasons

*tested on 3440x1440 res with RTX-5080 and i7-10700k

Design

Overlays

Standings

Classic overview of the session standings

DDU

All in one dashboard for the driver with many features

Relative

Competitor list centered on your car with opitional minimap, and customizable columns

Inputs

Often used pedals and steering visualization for driving consistency

Delta

Circular delta with changeable reference time and predicted lap time

Flags

High contrast two band banner for session and race control flags

Track Map

Handcrafted trackmap with support for multi class racing

Weather

All in one weather overlay that displays all data relative to your car and track position

Radar

Proximity radar with customizable distance guides and sticky alerts

Pit

Pit stop progress bar and digital 'PE' warning when entering/exiting pit road

Fuel Calculator

Shows all data relative to your fuel level etc.

Tire

Tire wear and service indicators in pit lane. (no live updates)
