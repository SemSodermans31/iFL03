Modern iRacing Overlays

Lightweight, polished, configurable

iFL03 is a continuation of the iRon project with a refreshed visual style, sensible defaults, and a friendlier setup experience via a built‑in Chromium (CEF) UI. It runs well on modest PCs and keeps dependencies to a minimum.

Download Install guide

What’s inside

New overlays

  • Delta, Flags, Weather
  • Track Map and Radar

Refined classics

Relative, Standings, DDU and Inputs get cleaner typography, better data, and useful options like averages, class colors and iRating hints.

Preview & UI

Preview mode fills data so you can place overlays off‑track. The CEF UI exposes common settings with live reload from config.json.

Overlay selection

Install & run

  1. Download the latest zip from Releases and extract anywhere writable.
  2. Launch iFL03. Start iRacing; overlays show when you’re in the car.
  3. Use Preview Mode to place overlays. Tweak options in the UI or config.json.
